# Document_Verification_Analysis
Document Verification Analysis project using Excel, featuring data cleaning, dashboard creation, KPI tracking, Pivot Tables, charts, and business insights. 

 The Document Verification Analysis project is designed to analyze and monitor the document verification process using Microsoft Excel. This project is based on a dataset containing more than 10,000 records, including applicant details, document information, verification status, submission dates, verification dates, processing times, risk levels, departments, states, cities, and approval details. The primary objective of this project is to transform raw data into meaningful insights that help improve operational efficiency and decision-making.

The project begins with a comprehensive data cleaning process. Missing values were identified and handled using appropriate techniques, duplicate records were removed, inconsistent data formats were standardized, and data quality issues were resolved. This step ensured that the dataset was accurate, reliable, and ready for analysis. Several Excel functions and techniques were applied to prepare the data for reporting and visualization.
